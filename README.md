# nmh4598.github.io

For full documentation visit [mkdocs.org](https://www.mkdocs.org).

## Commands
- Create a new project.
```
mkdocs new [dir-name]
```

- Start the live-reloading docs server.
```
mkdocs serve
```

- Build the documentation site.
```
mkdocs build
```

- Print help message and exit.
```
mkdocs -h
```
